# Embedded_Notes —— 嵌入式工程师成长笔记库

## 一、建立这个笔记库的目的

本笔记库用于系统记录我在 **嵌入式学习与工作过程中** 的所有核心知识、项目经验和问题解决过程，主要目标：

1. 构建完整的嵌入式知识体系结构
2. 形成个人长期可复用的技术库
3. 沉淀项目经验与踩坑经验
4. 提高面试竞争力
5. 为成为高级嵌入式工程师做积累
